\doxysubsection{Detailed Description}
Node that prints information about the position and velocity of a robot. 

\begin{DoxyAuthor}{Author}
Aurora Bottino 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
24/01/2023
\end{DoxyDate}
This node subscribes to the \char`\"{}/pos\+\_\+vel\char`\"{} topic and prints information about the robot\textquotesingle{}s position and velocity. It also computes the distance from the desired position and the average speed of the robot. The desired position is obtained from the parameters \char`\"{}des\+\_\+pos\+\_\+x\char`\"{} and \char`\"{}des\+\_\+pos\+\_\+y\char`\"{}, and the publish frequency is obtained from the parameter \char`\"{}frequency\char`\"{}.
